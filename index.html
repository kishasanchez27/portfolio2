BanasUno: Neighborhood Heat Awareness and Advisory

A web-based tool that helps residents of Davao City understand heat risk in their own neighborhoods and locate nearby health facilities. Uses official PAGASA heat levels to create an interactive, color-coded map with real-time risk visualization, AI-powered safety recommendations, and comprehensive data dashboards.

React Vite Node.js Express Supabase PAGASA API Open-Meteo Vercel Framer Motion
🌐 Live Demo
Key Features:
  • Color-coded heat map using PAGASA heat index classifications (Not Hazardous to Extreme Danger)
  • Interactive barangay selection with detailed zone information and nearby health facilities
  • AI-powered heat advisory with personalized safety recommendations
  • Comprehensive dashboard with KPIs, barangay tables, and historical trends
  • Route planning to health facilities with multiple travel modes
  • CSV data export for heat risk and historical trends
  • IoT sensor concept integration for future community-sourced data
  • Fully responsive design for mobile and desktop
